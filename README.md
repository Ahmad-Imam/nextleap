# NextLeap

## Welcome to NextLeap.

Live: https://kickstart-ahmadimam.vercel.app/

## Information

NextLeap is a job management application built in Next.js, Prisma and NeonDB.

## Frontend Instructions

1. Run npm install --legacy-peer-deps (Some package version conflict with React 19 and Next.js 15)
2. run npm run dev
3. Enjoy !!!

## Application Features

1. Create and track your job applications

2. After creating the job you can update its status to interview dates and finally selected or rejected

3. You can create customized cover letter for the job based on your profile settings and job details

4. Login via email or google to create or modify anything on the website.

5. You can create a resume based on your profile settings or customized settings. You can save the resume for later use.

6. In your dashboard you will find any upcoming interviews for the next 15 days.

7. You can switch the theme from light mode to dark mode anytime.
